# Innovation Strategy

Load the `mba:innovation` skill and apply it to the user's challenge.

## Steps

1. **Load the skill.** Read `skills/innovation/SKILL.md` for core frameworks and instructions.

2. **Clarify the innovation need.** Determine whether this is about new product development, customer problem reframing, innovation prioritization, or concept testing.

3. **Consult references as needed** from `skills/innovation/references/`.

4. **Apply the framework rigorously** and provide actionable recommendations.
